# My Account — App Android

## Comment générer votre APK (5 minutes, gratuit)

### Étape 1 — Créer un compte GitHub
- Allez sur **github.com**
- Cliquez **Sign up** → entrez email, mot de passe, nom d'utilisateur
- Confirmez votre email

### Étape 2 — Créer un nouveau dépôt
- Cliquez le **+** en haut à droite → **New repository**
- Nom : `myaccount-app`
- Cochez **Public**
- Cliquez **Create repository**

### Étape 3 — Uploader les fichiers
- Sur la page du dépôt, cliquez **uploading an existing file**
- Sélectionnez TOUS les fichiers du dossier MyAccountApp
  (glissez-déposez tout le contenu)
- Cliquez **Commit changes**

### Étape 4 — Télécharger votre APK
- Allez dans l'onglet **Actions**
- Cliquez sur le build en cours (roue verte ou orange)
- Attendez 3-5 minutes
- Cliquez sur **MyAccount-APK** pour télécharger

### Étape 5 — Installer sur Android
- Transférez l'APK sur votre téléphone
- Allez dans **Paramètres → Sécurité → Sources inconnues** (activer)
- Ouvrez le fichier APK → **Installer**
